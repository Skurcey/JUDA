# Vision de JUDA

## Pourquoi JUDA existe

JUDA est né d'une idée simple :

Créer un jeu où les joueurs fabriquent eux-mêmes leurs histoires.

Le but n'est pas de tuer un dragon.

Le but est de raconter, quelques jours plus tard :

"Tu te souviens quand le maire a disparu avec la caisse de la bibliothèque ?"

ou

"J'ai accusé le mauvais type pendant le procès..."

Ces histoires sont le véritable contenu du jeu.

---

# Notre philosophie

Nous préférons :

Les interactions
plutôt que
les statistiques.

Les choix
plutôt que
les classes.

Les conséquences
plutôt que
les récompenses.

Les histoires
plutôt que
les quêtes.

---

# Les cinq piliers

## 1. Le gameplay crée les histoires.

Chaque mécanique doit favoriser les interactions entre joueurs.

Si une fonctionnalité n'amène aucune anecdote...
elle n'a probablement pas sa place dans JUDA.

---

## 2. Les compétences sont cachées.

Le joueur ne connaît jamais son niveau exact.

Il découvre simplement de nouvelles possibilités.

Les autres joueurs doivent apprendre à vous connaître.

---

## 3. Le monde appartient aux joueurs.

Les métiers.

Les commerces.

Les grands projets.

La politique.

L'économie.

Tout doit être principalement animé par les joueurs.

---

## 4. Les conséquences sont sociales.

Perdre de l'argent.

Perdre sa réputation.

Perdre des alliés.

Être accusé.

Être innocenté.

Le jeu cherche les conséquences intéressantes plutôt que les punitions définitives.

---

## 5. Le serveur fait les règles.

Le LLM raconte.

Le serveur décide.

L'IA enrichit le monde.

Elle ne crée jamais de nouvelles règles.

---

# Notre règle d'or

Avant d'ajouter une fonctionnalité,
nous devons toujours nous demander :

"Quelle histoire cette mécanique permettra-t-elle de raconter ?"

Si la réponse est :

"Aucune."

Alors cette fonctionnalité n'a probablement pas sa place.

---

# Ce que JUDA ne sera jamais

- Un MMORPG classique.
- Un jeu où le grind est une fin en soi.
- Un jeu où les chiffres sont plus importants que les joueurs.
- Un jeu où la meilleure stratégie consiste à jouer seul.
- Un jeu où l'IA remplace les interactions humaines.

---

# Notre objectif

Faire rire.

Faire douter.

Faire négocier.

Faire trahir.

Faire pardonner.

Et surtout...

Faire raconter les parties.

---

BEEP BOP.

Une bonne partie de JUDA ne se termine pas
quand le joueur quitte le jeu.

Elle continue lorsqu'il raconte
ce qui lui est arrivé.
