# Le Monde

## Vision

JUDA se déroule dans une ville vivante.

Les habitants la construisent.

Les métiers la font fonctionner.

Les décisions politiques la transforment.

Le monde n'est jamais totalement terminé.

Il évolue avec les joueurs.

---

# Les lieux

Chaque lieu possède :

- une ambiance ;
- des habitants ;
- des objets ;
- des métiers ;
- des événements ;
- des actions disponibles.

Exemples :

- Place du marché
- Taverne
- Mairie
- Tribunal
- Bibliothèque
- Boulangerie
- Forêt
- Carrière
- Rivière
- Port
- Égouts
- Château

---

# Les bâtiments

Les bâtiments appartiennent généralement à un joueur, une organisation ou à la ville.

Ils peuvent :

- produire des ressources ;
- vendre des services ;
- accueillir des événements ;
- évoluer.

Un bâtiment raconte une histoire.

Une vieille forge abandonnée n'aura pas les mêmes possibilités qu'une forge florissante.

---

# Les régions

Le monde est découpé en régions.

Chaque région possède :

- ses ressources ;
- ses spécialités ;
- sa culture ;
- ses habitants ;
- ses événements.

Toutes les ressources ne sont pas disponibles partout.

Le commerce devient naturellement nécessaire.

---

# Les ressources

Le monde produit naturellement :

- Bois
- Pierre
- Argile
- Eau
- Minerais
- Poissons
- Céréales
- Fruits
- Champignons
- Gibier

Ces ressources sont ensuite transformées par les métiers.

---

# Les saisons

Les saisons modifient naturellement le monde.

Quelques exemples :

- certains fruits disparaissent ;
- les rivières peuvent geler ;
- les récoltes changent ;
- certaines professions deviennent plus rentables.

Le monde doit sembler vivant.

---

# Les événements

Des événements apparaissent régulièrement.

Exemples :

- Tempête
- Festival
- Incendie
- Épidémie
- Élections
- Inauguration
- Procès célèbre
- Visite d'un marchand

Les joueurs peuvent les influencer.

---

# Les grands projets

Le maire peut proposer un grand projet.

Exemples :

- Pont
- Remparts
- Université
- Port
- Aqueduc
- Bibliothèque
- Hôpital
- Théâtre

Chaque projet nécessite :

- des ressources ;
- des artisans ;
- du financement ;
- du temps.

Une fois terminé, il apporte un bonus permanent à la ville.

Mais il peut aussi devenir le théâtre de détournements de fonds, de corruption ou de scandales.

---

# Les objets du monde

Le monde contient :

- des objets utiles ;
- des objets de collection ;
- des objets décoratifs ;
- des objets humoristiques ;
- des objets mystérieux.

Le joueur ne sait jamais immédiatement si un objet est réellement utile.

Une vieille chaussure peut ne rien faire...

...ou entrer dans une recette secrète.

L'exploration récompense la curiosité.

---

# Les secrets

Le monde cache :

- des métiers oubliés ;
- des recettes improbables ;
- des livres rares ;
- des passages secrets ;
- des personnages insolites ;
- des événements extrêmement rares.

L'objectif est que chaque joueur découvre quelque chose que les autres ignorent.

---

# Philosophie

Le monde de JUDA ne doit jamais donner l'impression d'être figé.

Même sans la présence du joueur, la ville continue de vivre.

Des bâtiments apparaissent.

Des commerces ferment.

Des journaux sont publiés.

Des rumeurs circulent.

Le joueur ne visite pas un décor.

Il rejoint une histoire déjà en cours.

Monde en constante evolution pour les X semaines que dirent la partie...c est plus qu un monde c est un personnage !
